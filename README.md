# Purdy Burdy Books website

A simple one-page static website designed for GitHub Pages.

## Publish on GitHub Pages

1. Upload `index.html`, `styles.css`, and the `assets` folder to the root of your repository.
2. Open the repository's **Settings**.
3. Choose **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. When GitHub provides the temporary web address, open it and confirm the page works.
7. Return to **Settings > Pages**, enter `purdyburdybooks.com` under **Custom domain**, and save.

The email address in the footer is a placeholder. Change it in `index.html` if needed.
